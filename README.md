# Tweet_Sentiment

Objectives: 
  1. Get the most frequent entities from the tweets.
  2. Find out the sentiment/polarity of each author towards each of the entities.

Steps involved:
 1. Loading the json file
 2. Translating dataset to English language using googleTrans
 3. Data Preprocessing
 4. Using Bag of words to extract count of most frequent entities
 5. Using word2vec to get contextual based word embeddings
 6. Clustering the words based on sentiment using KMeans clustering
